# Kanban MVP

Aplicacion Next.js con una sola pizarra Kanban:

- 5 columnas fijas y renombrables
- Tarjetas con titulo y detalle
- Agregar, eliminar y mover tarjetas por drag-and-drop
- Sin persistencia

## Scripts

- `npm run dev`: servidor local
- `npm run lint`: lint
- `npm run test:unit`: pruebas unitarias (Jest)
- `npm run test:e2e`: pruebas de integracion (Playwright)
